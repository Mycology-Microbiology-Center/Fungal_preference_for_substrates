# Fungi-substrate association patterns in boreal forests
- This repository contains the code and resources used for analyzing fungal biodiversity, assembly and dissimilarity in community composition across substrates  based on PacBio sequencing. This repository explores the influences of fungal-specific selection and dispersal process on fungal biodiversity in substrates.
# Scripts
- **Organised Code.R** – scripts for generating Figures 1–5  
- **Organised Code2.R** – scripts for generating Figures 5AB, S1, and remaining analysis
